# gkMovie 🎬

gkMovie is a movie discovery application built with React and TMDB API.

Users can explore movies, search for titles, view detailed movie information, watch trailers, browse cast members, and manage their favorite movies.

## 🌐 Live Demo

https://www.gkcode.net/gkmovie/

## Features

- Browse now playing, popular, upcoming and top rated movies
- Discover movies by categories
- Search movies with pagination support
- View detailed movie information
- Watch movie trailers with modal player
- View movie cast members
- Add and remove movies from favorites
- Persistent favorite system using LocalStorage
- Global state management with Context API
- Responsive design for mobile and desktop devices

## Technologies

- React
- React Router
- Context API
- Vite
- Tailwind CSS
- Axios
- Swiper.js
- Radix UI
- TMDB API
- LocalStorage
