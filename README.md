# gkMovie 🎬

gkMovie is a movie discovery application built with React.
Users can explore movies, view movie details, watch trailers, and browse popular actors.

## Features

- Browse now playing, popular, upcoming and top rated movies
- View detailed movie information
- Watch movie trailers with a modal player
- View movie cast members
- Responsive design for mobile and desktop devices

## Technologies

- React
- React Router
- Tailwind CSS
- Axios
- Swiper.js
- Radix UI
- TMDB API
